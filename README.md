# xdg-tui
A simple TUI that allows a user to more intuitively view their XDG settings

Currently this will show XDG_ variables, MIME defaults, and Applications.

This started out as a quality-of-life project. Since I'm porting it to Rust, I figured that other people might find some use in it.
